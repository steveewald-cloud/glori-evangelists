## 1. The decision to be made

Whether each GLORi Evangelist is engaged as an **independent contractor (1099)** or an **employee (W-2)** — and whether that answer should differ **by rep type and program stage** (e.g., early field reps vs. a salaried core team).

This is a legal/tax classification decision, not a branding or pay-design decision. It is governed by federal (IRS common-law control test, DOL economic-reality test under the FLSA) and **state** law — and several states (notably California's ABC test) are materially stricter than the federal baseline. Counsel must run the test that applies in each state where a rep sells.

**Scope note (carried from program guardrails):** This brief covers Evangelists only — **paid GLORi field sales reps who earn cash commission + residual.** It does **not** cover Ambassadors (existing customers who refer peers and are rewarded in account credit / a Bless, never cash). Ambassadors are not workers and are out of scope for classification. Insurance and financial product lines are **out of scope** for Evangelists (licensing) and are not part of this analysis.

## 2. What the comp model already fixes (and why it matters here)

The Evangelist plan is **fully locked as Model D** (see `COMP-PLAN-MODEL-D.md`). Two features of it are directly relevant to classification because they speak to **financial control** and **economic dependence** — factors in both the IRS and DOL tests:

- **Pure commission, no base, no draw.** Core rate is **20% of MRR for an account's months 1–12**, then a **5% residual for months 13–60 (years 2–5)**, both **employed-only** (pay stops when the rep leaves; no clawback of amounts already paid). A **configurable draw exists in software but defaults OFF.**
- **Opportunity for profit/loss tied to the rep's own effort.** Per retained, employed, residual-qualified account: Year 1 = **$240 / $960 / $2,400** (Starter / Growth / Pro); residual/yr = **$60 / $240 / $600**; 5-year lifetime = **$480 / $1,920 / $4,800**. Residual **qualifies only in a year the rep produces ≥ 50 new accounts** (re-qualify annually). Stacking one-time milestone bonuses: **$1,000** (10 sites, month 1), **$2,500** (25 in 3 months), **$10,000** (100 in 12 months) — **$13,500** if all three hit.
- The entire plan is **"subject to management change at any time."**

**Why this matters:** commission-only (or draw-plus-commission) pay is *consistent with* independent-contractor status **only if the rest of the relationship keeps control light.** Pay structure alone does not settle classification. If GLORi dictates hours, scripts, territory in a controlling way, mandatory meetings, exclusivity, or close supervision, a worker can be a **W-2 employee regardless of being paid 100% commission.** The draw toggle defaulting OFF is helpful for an IC posture; turning it ON for a given rep is a fact that leans (modestly) toward employee-like economic dependence and should be reviewed with counsel.

**Faith-posture note:** prayer/vision are offered consensually and are never a condition of a sale or of engagement. Nothing in the classification paperwork should convert a consensual faith practice into a mandated job duty.

## 3. Trade-offs to weigh

| Dimension | 1099 Independent Contractor | W-2 Employee |
|---|---|---|
| **Autonomy / control** | Rep controls how/when/where they work; GLORi specifies the *result*, not the *method*. Light control is required to defend the classification. | GLORi may direct methods, hours, scripts, mandatory meetings, territory, tools, supervision. |
| **Cost (fixed)** | Lower fixed cost. No employer payroll tax, no benefits, no unemployment/workers'-comp premiums (subject to state rules). Pay is variable with production. | Higher fixed cost: employer FICA (~7.65%), FUTA/SUTA, workers' comp, benefits, admin/payroll. |
| **Control GLORi can exert** | Limited. Detailed direction erodes the classification. Can set deliverables, standards, brand/compliance rules — but not day-to-day method control. | Broad. Can require training, schedules, reporting cadence, performance management, at-will discipline. |
| **Benefits** | None provided (rep sources own health, retirement, etc.). | Eligible for benefits, PTO, employer retirement, protections under wage/hour and other statutes. |
| **Tax mechanics** | Rep gets **1099-NEC**; responsible for self-employment tax and own withholding/estimated payments. GLORi collects **W-9**. | GLORi withholds income tax + employee FICA; issues **W-2**; collects **W-4** and **I-9**. |
| **Misclassification risk** | **The principal risk.** If control/economic-dependence facts look employee-like, exposure includes back payroll taxes, unpaid overtime/minimum wage, penalties, interest, benefits, and state-agency/private actions. State tests (e.g., CA ABC) can reclassify even where the IRS test would not. | Very low classification risk. Trade-off is cost and reduced flexibility, plus full wage/hour compliance obligations (overtime, minimum wage, expense reimbursement). |
| **Termination / stopping pay** | Governed by the IC agreement's term/termination clause. Model D residual is already employed-only and stops at departure. | At-will (per offer letter/handbook), subject to final-pay and wage-law rules. |
| **Expense handling** | Rep bears own expenses (documented in an expense policy); reinforces IC economics. | Employer typically reimburses business expenses; some states mandate it. |

## 4. Likely path (for counsel to confirm — not a decision)

A **staged / mixed** approach is the pattern most consistent with how the program is built, **subject to counsel's state-by-state review:**

- **Early field reps (commission-only, self-directed, own hours/territory, draw OFF):** most naturally fit a **1099 independent-contractor** posture — provided GLORi keeps control light and does not impose employee-like direction. This matches the Model D "no base, no draw, pure commission, opportunity for profit/loss" economics.
- **A salaried / core team (managers, trainers, reps under close direction or with draw ON, or anyone GLORi needs to control day-to-day):** more naturally fit **W-2 employee** status. If GLORi wants schedules, mandatory processes, exclusivity, or supervision, W-2 is the safer and often the correct classification.
- **Watch the drift:** a rep can *start* looking like a contractor and *become* an employee in substance as GLORi adds control (mandatory meetings, scripts, quotas enforced as discipline, exclusivity, provided equipment, draw ON). Re-test classification when the relationship changes. **Substance over label** — the agreement title does not control; the facts do.
- **State overlay:** in ABC-test states, the "B" prong (work outside the usual course of the hirer's business) is hard for a sales rep selling GLORi's core product to satisfy. Counsel must confirm whether 1099 is even available in those states before offering it.

## 5. Paperwork sets — outline / skeleton only

Each rep must have a **complete, signed paperwork set before selling.** Below are **bullet skeletons, not final legal text.** Counsel must draft/finalize every item. Both sets should carry the "subject to management change at any time" posture and reference the locked Model D comp terms rather than restating pay in a way that could drift out of sync.

### 5A. Independent Contractor Agreement set (1099)

- **Independent Contractor Agreement**
  - Parties, effective date, scope of services (selling GLORi to prospects)
  - Explicit IC status; rep controls method/means; no employee benefits; no withholding
  - Commission-only pay referencing Model D (20% yr-1 / 5% residual yrs 2–5, employed-only, residual ≥50-accounts/yr gate, 5-yr per-account cap, stacking bonuses); draw defaults OFF
  - "Subject to management change at any time"; no guaranteed income
  - Term, termination (either party), effect of termination on residual (stops at departure; no clawback of amounts already earned/paid)
  - Rep responsible for own taxes, tools, expenses, insurance
  - Compliance rails: honest selling, faith posture consensual/non-coercive, **no insurance/financial product lines**, Ambassador vs. Evangelist boundary (rep pay is cash commission and never flows through Ambassador account-credit rewards)
  - Confidentiality; no authority to bind GLORi beyond defined scope; dispute resolution / governing law
- **IP Assignment / Work-for-hire assignment** — assigns any materials, leads data, and work product created for GLORi to GLORi
- **Non-solicitation** (customers and other reps) — scope/duration to be set by counsel and checked against state enforceability; **avoid a broad non-compete** in states that limit or ban them
- **Expense policy / acknowledgment** — rep bears own expenses; documents the IC economics; reimbursement (if any) narrowly defined
- **Form W-9** — collected before first payment; drives 1099-NEC reporting

### 5B. W-2 Offer Letter set

- **Offer Letter**
  - Position, start date, reporting line, **at-will** statement
  - Compensation: commission per Model D and/or salary/base as applicable; if draw is used, state it explicitly; pay schedule
  - Benefits eligibility summary (referencing plan docs, not guaranteeing terms)
  - Contingencies (I-9 work authorization, background check if any)
  - "Subject to management change at any time" for the variable comp
  - Same compliance rails as 5A: consensual faith posture, insurance/financial lines out of scope, Evangelist/Ambassador boundary
- **IP Assignment** — employee assigns work product / inventions created in scope to GLORi
- **At-will + Handbook acknowledgment** — confirms at-will status and receipt of policies (conduct, anti-harassment, compliance, faith-posture guardrails)
- **Form I-9** — employment-eligibility verification (with timing rules counsel will specify)
- **Form W-4** — federal withholding election (plus any state equivalent)

## 6. Open questions for counsel

1. In each state where reps will sell, is **1099 available** for a GLORi sales rep under that state's test (esp. ABC states)?
2. Does any planned **control** (mandatory meetings, scripts, quotas-as-discipline, exclusivity, provided equipment, **draw ON**) push a nominal 1099 rep into **employee** status?
3. What **non-solicit** scope is enforceable per state; should non-compete be avoided entirely?
4. **Wage/hour** exposure for any W-2 commission rep — overtime, minimum-wage true-up, expense-reimbursement mandates?
5. Reporting/onboarding **timing** rules (W-9 before first pay; I-9/W-4 before/at start).
6. How to keep each paperwork set **in sync with Model D** as management changes the plan, without re-drafting agreements each time.
